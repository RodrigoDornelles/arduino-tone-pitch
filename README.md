this library is very important for those who prefer to have the buzzer tuned correctly for their musical project, it is fully compatible with other third party libraries.

## How to Use ##
### Simple Bip ###
```C
#include <Arduino.h>

#define TONE_USE_INT
#define TONE_PITCH 440
#include <Pitch.h>

void setup(){}

void loop()
{
  tone(8, NOTE_C4, 500);
  delay(1000);
}
```
